클래스규칙
채널별 Class 부여 규칙
  • wrap 클래스에 class로 부여 함. <div class="wrap page-xxx">
  • 홈페이지(메인): .page-main
  • 한화펀드: .page-fund
  • 인사이트: .page-insight
  • 공시: .page-notice
  • 회사소개: .page-company
하위 화면별로 구분 Class 부여 규칙
  • contents 클래스에 class로 부여 함. <div class="contents fund-xxx">
  • 펀드 메인: .fund-main
  • 펀드 검색 결과: .fund-search
강제 클래스
경우에 따른 Show/Hide
  • Mobile에서만 보여야 하는 경우: .mob-only
  • PC에서만 보여야 하는 경우: .pc-only
