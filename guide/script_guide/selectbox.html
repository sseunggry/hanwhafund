    Options

    Attributes
    Property Description Type Default
    type default, account

    기본(default), 최근이용계좌(account)    		 String default
    className 레이어에 추가 될 클래스 네임 String null
    label 셀렉트 박스 제목 - title로 제목을 제공하지 않을 경우 string null
    minWidth 최소 사이즈 지정(PC 대응) Number 0
    pos 셀렉트 레이어 노출 위치 지정
    "bottom-left" or "bottom-right"    		 String bottom-left


    Methods

    Attributes
    Name Description parameter
    setValue() 리스트 중 selected, fui-selected 속성이 부여된 요소를 찾아 셀렉트박스 상태 업데이트 int, Element, "",
    reset() label 문구로 셋팅 없음 
       var selectbox = document.querySelector('[fui-selectbox]');

// setValue()
  
   // 리스트 중 selected, fui-selected 속성을 찾아 상태 업데이트, 속성이 없을 경우 선택자는 0번째 인덱스
   MOui.selectbox(selectbox).setVAlue()
  
   // n 번째 인덱스로 selected 요소 지정
   MOui.selectbox(selectbox).setVAlue(n)
  
   // 전달된 Element로 selected 요소 지정
   MOui.selectbox(selectbox).setVAlue(Element)


    Events

    Attributes
    Name Description Example
    change 옵션 변경 시 발동 
    // orignal
var selectbox = document.querySelector('[fui-selectbox]');
selectbox.addEventListener('change', function(e){
  console.log(e.detail);
})



 // jQuery
 $('[fui-selectbox]').on('change', function(e){
   console.log(e.detail);
 })


  // e.detail[0] 셀렉트박스 전체 객체 정보
  // e.detail[1] 셀렉트박스 선택 요소 정보 (index, target)

    Attributes
    Property Description Type Default

    Attributes
