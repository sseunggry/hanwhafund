알럿모달 & 컨펌모달

PREVIEW MARKUP





모달닫음 후 적용할때 closecallback 필수 사용

예) 약관 비동의
$plugins.uiModalClose({ id:'모달아이디', remove:true, closecallback: function () {ih.ui.modalConfirm( {result:'N'})}});
예) 페이지이동
$plugins.uiModalClose({ id:'모달아이디', remove:true, closecallback: function () {MOui.step('[fui-step]').showSlideStep();})}});






Options

Attributes
Property Description Type Default
full 풀페이지여부 Boolean false
remove 모달제거 Boolean false
ps 모달생성방향 String "center"
callback modal이 열리면 실행되는 콜백함수 function false
closecallback modal이 닫히면 실행되는 콜백함수 function false
words 알럿 내용 글 Boolean false
btntxt1 버튼1 Boolean false
btntxt2 버튼2 Boolean false
btntxt3 버튼3 Boolean false
half 모달 최소 높이 커스텀 Boolean false
autofocus 모달닫힌 후 포커스이동처리 Boolean true
endfocus 모달닫힌 후 포커스이동 id 값 (autofocus 값이 true 일때만 적용) string null
