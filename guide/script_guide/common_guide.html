※ 이슈 사항

History back

  • 1. 모니모, 삼성화재 앱(안드로이드) [새 창으로 띄우는 페이지들]
    front(메인)에서 이동하는 첫 페이지에서 back() 시도 시 브라우저 닫고 바닥 페이지인 front 페이지로 이동
    step 페이지를 제외한 일반 페이지에서는 기본 back() 동작
    step 페이지(첫 페이지 제외)에서는 이전 스텝으로 이동
    step 프로세스 완료 후 back() 시도 시 페이지 닫고 front(메인)으로 이동

  • 2. 그 외 기타 현재 페이지 이동 페이지들
    front(메인)에서 이동하는 첫 페이지에서 back() 시도 시 index.html 로 이동(History back 아님)
    step 페이지를 제외한 일반 페이지에서는 기본 back() 동작
    step 페이지(첫 페이지 제외)에서는 이전 스텝으로 이동
    step 프로세스 완료 후 back() 시도 시 index.html 로 이동(History back 아님)

  • 3. pc, mobile 이슈
    페이지 앞으로 가기, 뒤로가기의 구분 없음
