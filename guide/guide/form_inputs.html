텍스트 필드
Input Basic
  • 텍스트 필드의 기본적인 형태
  • label에는 항목명이 표시되고, placeholder에는 힌트 텍스트가 표시됨
  • placeholder가 label로 이동하는 상호작용이 일어나지 않음
  • 필수입력 .bullet-must는 .label-txt상이 아닌 단독으로도 사용가능(모듈 > 리스트 참조)
PREVIEWMARKUP
Input Secondary
  • placeholder가 label인 형태로 힌트 텍스트를 별도로 사용하면 안됨.
  • 사용자가 텍스트 필드에 포커스가 들어가면 placeholder의 텍스트가 label로 이동하는 상호작용이 일어남
  • 텍스트 필드에 포커스가 들어가면 .form-wrap에 .is-focus가 들어감. 텍스트 없는 상태에서 포커스 아웃 될 경우 .is-focus 삭제됨.
PREVIEWMARKUP
Input Complex
  • Basic에 unit, button, image를 각각 조합해서 사용하는 형태
  • .inp-wrap에 .has-button을 추가함.
